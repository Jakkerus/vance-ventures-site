# Vance Ventures

Official website for Vance Ventures, a private holding company focused on ownership, execution, and long term value.

## Overview

Vance Ventures builds and grows businesses through disciplined ownership and operational execution. The company is focused on developing a portfolio of durable businesses across industries.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- AWS Amplify (hosting)
- Route 53 (DNS)

## Development

Install dependencies:

npm install

Run locally:

npm run dev

Open http://localhost:3000

## Deployment

The site is deployed via AWS Amplify using the main branch.
